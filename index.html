Introduction

HTML means Hyper Text Markup Language. Hyper Text is a bunch of documents that are linked using hyperlinks. It is the standard language used for displaying contents in webpagees. It is the background for creating a websites and the basics for learning how to build a websites for beginners. It uses tag and atrributes to define the meaning and structure of a webpage. HTML is not a programming language because it does not have the ability to do dynamic functionality on a webpage unlike other programming language that uses sripts e.g. Javascript. The filename extension of Html is .html or .htm which means all html files should end with either .html or htm. The HTML pages can be view using web pages such as firefox, chrome, operamini etc. it reads the HTML file and display the contents to the users.

Tags

  • < html > Tag :- < html > tags are hidden keywords and commands that can only be seen by the web browsers. it tells the browser how to struture the contents of a webpage. HTML Tags has an opening tag < html > and a closing tag < /html >. The opening tag also called the start tag is always placed at the begining of the documents. The closing tag also know as the end tag is always placed at the end of the documents. The closing tag has a slash after the first bracket unlike the opening tag.

  • < head > Tag :- The < head > tag is after the opening html tag. It defines the head section of the HTML document. It contains the non-visual elements of the webpage that are only visible to the web browsers and alos specific information related to the documents.


    Syntax:-
    < head >
    < title > < /title >
    < meta >
    < link >
    < /head >

    • < title > Tag :- This is placed between the opening and closing head tag. It describes the title of the webpage and also used by search engines for search listings. It always appears at the top of the web browser of the webpage.

    • < meta > Tag :- It is an empty tag i.e. It has no closing tag. ( NOTE :- Empty Tags are tags that do not have a closing tag). It provides additional information about a webpage within its attributes (Attributes will be explained later).

    • < link > Tag :- This is also an empty tag. it is mostly useed to link a CSS file to an HTMl documents. It shows the relationship between a document and its external file.

  • < body > Tag :- It comes after the closing head tag. it is the main part of an HTML documents. It contains all the visual part of the webpage that is shown to the user. They includes paragraph, images, list, links, tables, break etc

    NOTE: AN HTML DOCUMENT HAS ONLY ONE HTML, HEAD AND BODY TAG.

    • < p > Tag :- It is used to create a paragraph on a webpage. The webbrowser will add an new line before and after a paragraph automatically. It is not an empty tag.

    • < html headings > Tag :- HTML has 6 headings which are ranked from the most important to the least important. They are not an empty tags. The HTML headings are :- < h1 > , < h2 > , < h3 > , < h4 > , < h5 > , < h6 > .

    • < br > Tag:- It is use to create a line page in a text. it is an empty tag. It is often used when writing a poem or address.

    • < hr > Tag:It is usede to add an horizonatal line to seperate contents in a page. it also defines a thematic break in an HTML page (e.g A shift in topic).

    • < a > : This is called an archor tag. An Anchor tag is used to define hyperlink for linking one webpages, emails, files e.t.c together. it is used by Search Engines to send user to specific section of the webpage. The Anchor tag requires an href attributes. An href attributes contains URL to send user when they click on the link.

      < a href = "www.google.com" > Go to Google < /a >

      This is the way an anchor tag is written, it also have other attributes e.g name, target etc. Go to Freecodecamp to learn more.

    • < img > Tag: This is used to embed an image to an HTML pages. It is a self-closing tag .i.e it has no closing tag. It has two required attributes.

      • src: It defines the path of the image to be embedded.

      • alt: It defines the text that the user will see if the image is not showing

      < img src = "img.cup.jpg" alt = a cup >

    • list items: it is used to group a set of related items in a semantic and orderly manner in a page. We have the ordered and ordered list.

      • < ol > :It is called an ordered list. It defines an ordered list. An ordered list is used to create a related items in alphabetical order or in numerical order. An ordered list is created using the opening < ol > followed by both opening and closing list items < li > , < /li > and the content in between, then the closing ordered list follows < /ol > .

        • < ol >
        < li > Travelling < /li >
        < li > Chatting < /li >
        < li > Coding < /li >
        < /ol >

        The above example will show in your browser this way:

        1. Travelling
        2. Chatting
        3. Coding

      • < ul > :It is called an Unordered list.It is used to defined list of related items using bullets. An unordered list is created using the opening < ul > followed by both opening and closing list items < li > , < /li > and the content in between, then the closing ordered list follows < /ul > .

        • < ul >
        < li > Travelling < /li >
        < li > Chatting > < /li >
        < li > Coding > < /li >
        < /ul >

        The above example will show in your browser this way:

        • Travelling
        • Chatting
        • Coding

Elements

HTML Elements are the building blocks of webpages which defines the whole of HTML documents. They are used to add semantics and meaning to webpage. HTML elements contain an opening tag, the contents and a closing tag. Elements can also contain attributes

Technically, an element is a collection of start tag, atributes,end tag, content between them.

< p >

This is a paragraph

 < /p >
< h1 >

This is a first heading

< /h1 >

Block and Inline Elements

  • Block Elements:: Block elements are elements that always start on a new line, they do not allow other element to sit beside them and it takes all the full width avaliable. Block elements are <p > ,all Html headings, <form > , <hr > etc.

  • Inline Elements: Inline elements do not start on a new line,they allow all other element to sit beside the, and it only takes up as much width as necessary.They include a, br img etc

Attributes

Attributes provides more infomation about an HTML elements. All HTML elements can have attributes, they are always written at the beginning of an tag.
